This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).
Kelegam-Tech is a modern web application designed to showcase a polished user interface and developer experience, leveraging Next.js, React, TypeScript, Tailwind CSS, and Shadcn/UI components.
## Key Technologies
* **Next.js (App Router)**: Used for server-side rendering and static site generation.
* **React & TypeScript**: Employed for building robust and type-safe user interfaces.
* **Tailwind CSS**: Utilized for utility-first styling with a custom theme, including specific colors and animations.
* **Shadcn/UI**: A collection of accessible and customizable UI components built on Radix UI and Tailwind CSS.
To add new Shadcn/UI components, use the command `npx shadcn-ui@latest add <component-name>`.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load fonts. The primary fonts used are `Inter` for sans-serif text and `Space Grotesk` for display headings, loaded via `next/font` and configured through CSS variables.
